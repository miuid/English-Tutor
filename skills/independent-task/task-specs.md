# QCAA task specifications (Year 8, extendable to 9–12)

Source: reaserch.md (QCAA Year 8/9 assessment techniques & conditions).

## Year 8 conditions

| Technique | Mode | Length | Time (assessment mode) |
|---|---|---|---|
| Short response | written / spoken / multimodal | written 50–200 words per item (≤600 total); spoken ~1–2 min | within exam block |
| Extended response | written / spoken / multimodal | written **400–600 words**; spoken/multimodal 2–4 min | over an extended period |
| Examination | written, supervised | short 50–200/item; extended 400–600 | up to **70 min + 10 min planning** |

## Year 9 conditions (for later)

| Technique | Length | Time |
|---|---|---|
| Short response | written 100–300 words/item (≤800 total) | — |
| Extended response | written 600–800 words; spoken 3–5 min; multimodal 4–6 min | — |
| Examination | as above | up to 90 min + 10 min planning |

## Analytical task formats (Year 8)

Poetry analysis blog post · analytical essay · explanation · review. Focus: explain how text structures and language features (metaphor, imagery) shape meaning.

## Analytical task formats (Year 9, for later)

Critical literary essay · comparative essay · website critique · preface anthology. Focus: evaluate how representations reflect context and position readers.

## Copyright-safe stimulus rules
- Use public-domain texts (pre-1955 AU / classic poetry & prose) or generate an original passage.
- Never paste in a school's copyrighted set text; refer to it by name and have the student supply their own copy if analysing it.
