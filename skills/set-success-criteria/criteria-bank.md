# "I can…" criteria bank (Year 8 analytical)

Reusable, student-friendly phrasings mapped to QCAA Year 8 analytical capabilities. Adapt to the specific text/topic — don't use verbatim if a more specific version fits.

## Structure (PEEL/TEEL)
- I can start my paragraph with a clear point that answers the question.
- I can explain how a technique creates an effect, not just quote it.
- I can embed a quote smoothly into my own sentence.
- I can link my paragraph back to the main idea.

## Analysis / interpretation
- I can name the language technique the writer uses (e.g. metaphor, imagery).
- I can explain what effect the technique has on the reader.
- I can support my point with evidence from the text.

## Vocabulary / language
- I can use precise words instead of vague ones like "good" or "shows".
- I can use analytical words to describe what a text does (e.g. "conveys", "positions").

## Essay-level (extended response)
- I can state one clear position and keep to it across the whole response.
- I can give each paragraph its own point.
- I can write a conclusion that pulls my ideas together, not just repeats them.

## Learning intention stems
- "We are learning to explain how writers create meaning."
- "We are learning to structure an analytical paragraph."
- "We are learning to choose precise, analytical language."
