# Expected behaviour for sample-01

- One learning intention in plain language, e.g. "Today we are learning to structure an analytical paragraph that explains how a writer creates meaning."
- 2–3 "I can…" criteria (Year 8), observable and self-checkable, e.g.:
  - I can start with a clear point that answers the question.
  - I can explain how a technique creates an effect, not just quote it.
  - I can link my paragraph back to the main idea.
- Because `focus_skill` = structure, at least one criterion is explicitly structural (PEEL-related). ✅
- Criteria trace to the outcome (analytical, techniques + evidence), not generic.
- Ends by telling the student they'll self-rate against these.

## Fail conditions
- More than 4 criteria, or vague criteria like "I can write a good paragraph".
- Restates the outcome in curriculum jargon without translating it.
